# 🚀 部署到 Railway 完整教程

## ⏱ 总耗时：约 10 分钟

---

## 📋 第 1 步：注册 Railway

1. 打开 https://railway.app
2. 点击右上角 **「Login」**
3. 选择 **「Login with GitHub」**
4. 授权登录

---

## 📋 第 2 步：连接项目

1. 登录后进入 Railway 首页
2. 点击 **「New Project」**
3. 选择 **「Deploy from GitHub repo」**
4. Railway 会列出你的 GitHub 仓库，找到 **`questionnaire-filler`**
5. 点击选中它

---

## 📋 第 3 步：自动部署

Railway 会自动：
- ✅ 检测到这是一个 Node.js 项目
- ✅ 运行 `npm install`
- ✅ 启动 `npm start`
- ✅ 生成一个域名（类似 `xxx.railway.app`）

等 1-2 分钟，看到绿色 ✔ 就部署好了！

---

## 📋 第 4 步：访问你的网站

部署成功后，点击项目页面上的 **域名链接**（类似：`https://questionnaire-filler-xxx.railway.app`）

---

## 🎉 网站功能

| 功能 | 说明 |
|------|------|
| 🔐 注册/登录 | 新用户注册即送每日免费次数 |
| 🆓 每日免费 3 次 | 每天登录自动补充 |
| 💰 200份 = 18元 | 购买订单 → 微信扫码付款 → 管理员确认 |
| 🔍 分析问卷 | 粘贴问卷星链接自动解析题目 |
| ✏️ 每题自定义 | 选择想要的答案 |
| 🚀 批量提交 | 1-500 份实时进度 |
| 🌐 随机IP | 每份提交使用不同的中国IP地址 |
| 🛡️ 管理员后台 | 管理订单、充值、设置 |

---

## 👤 默认管理员

- 用户名：**admin**
- 密码：**admin123**

> ⚠️ 上线后请立即修改管理员密码！

---

## 💰 收款设置

管理员登录后 → 点「🛡️ 管理」→「系统设置」：

1. **payment_qrcode**：填写你的微信收款码图片URL
2. **payment_instructions**：填写收款说明文字
3. **price_per_200**：每200份的价格（单位：分，1800 = 18元）
4. **daily_free_quota**：每日免费次数
5. **min_buy_amount**：最少购买份数

---

## 🔧 手动充值

管理员后台 →「手动充值」→ 输入用户名和次数即可直接送次数

---

## 📊 数据存储

- 所有数据存储在 `data/` 文件夹（JSON 文件）
- Railway 免费版重启后数据不丢失（使用持久化卷）
